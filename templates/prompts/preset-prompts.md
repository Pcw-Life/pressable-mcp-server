# Preset Prompts for Pressable MCP

Copy and paste these into your AI assistant for quick results.

### General Management
- "List all my Pressable sites and highlight those that are currently in 'staging' mode."
- "Get the account activity logs for the last 24 hours and summarize any site deletions or creations."
- "What is the current storage usage for site 'my-web-site'?"

### Technical Operations
- "Run a WP-CLI command on site 'site-id-123' to deactivate all plugins and then reactivate only 'akismet'."
- "Purge the edge cache for 'production-site' and then verify the cache status."
- "Get the latest PHP error logs for 'blog-site' and tell me if there are any 'Fatal Error' occurrences."

### User & Security
- "Show me all WordPress users for 'client-site' and tell me who has the 'administrator' role."
- "Check all my sites for plugin security alerts. If any are found, list the site name and the specific plugin."
- "Generate a time-limited phpMyAdmin URL for 'database-site' so I can inspect the tables."

### DNS
- "List all DNS zones and find the records for 'example.com'."
- "Add an A record for 'subdomain.example.com' pointing to '1.2.3.4' in the 'example.com' zone."
