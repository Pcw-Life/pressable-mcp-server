# Contributing to Pressable MCP Server

Thank you for your interest in contributing! We welcome all kinds of contributions, from bug reports to new tools.

## How to Contribute

1.  **Report Bugs**: Use GitHub Issues to report bugs. Provide as much detail as possible, including steps to reproduce.
2.  **Submit Tool Ideas**: If you want a new Pressable API endpoint added as a tool, open an issue.
3.  **Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or fix.
    - Submit a PR with a clear description of the change.
    - Ensure your code follows the existing style and is well-documented.

## Development Setup

1.  Clone your fork.
2.  Install dependencies: `npm install`.
3.  Create a `.env` file with your Pressable API credentials.
4.  Run the server: `npm start`.

## Code of Conduct

Please be respectful and professional in all interactions within this project.
